# Download and launch
**On Debian-based distribution:**
```console
sudo apt install gcc
```
```console
sudo apt install libncursesw5-dev
```
```console
sudo apt install git
```
```console
git clone https://github.com/cla1rvoyant/clair-snake.git
```
```console
cd clair-snake
```
```console
gcc snake.c -o snake -lncurses
```
```console
./snake
```
